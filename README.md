## 🚀 Featured Projects

### 🛰️ [Intelligent Orbital Debris Removal Planner](https://github.com/KaramQ6/debris-removal-planner)
A dual-layer AI architecture designed to optimize propellant-constrained orbital debris collection in space:
1. **RL Core Agent**: A Reinforcement Learning model (MaskablePPO) that learns to sequence debris collection in a 3D Keplerian environment, minimizing total Delta-V.
2. **RAG Operational Advisory**: A Retrieval-Augmented Generation system providing mission teams with instant, grounded advice indexed from NASA/ESA debris mitigation guidelines.
* **Tech Stack**: Python, Stable-Baselines3, Gym/Gymnasium, PyTorch, BM25 RAG, Orbital Physics Simulation, Pytest.

### 💼 [Personal Portfolio](https://github.com/KaramQ6/Karam)
Karam's official developer portfolio website, featuring a modern glassmorphic interface and custom interactions:
* **Features**: Dynamic bento-grid layout, fluid scroll-driven animations, responsive UI/UX, and an integrated booking scheduling engine.
* **Tech Stack**: React, GSAP, Tailwind CSS, Vite, Vercel.

### 🐍 [agentflow](https://github.com/KaramQ6/agentflow)
An async-first, modular multi-agent AI pipeline framework designed with a simple, developer-friendly decorator API.
* **Features**: Light-weight, async-native execution, structured outputs using Pydantic, and available as a package on PyPI (`agentflowkit`).
* **Tech Stack**: Python, AsyncIO, Pydantic, PyPI.

### 🤖 [AI Crew Studio](https://github.com/KaramQ6/ai-crew-studio)
A collaborative multi-agent orchestration platform designed for managing autonomous AI agent crews:
* **Features**: Real-time streaming output, collaborative AI workflows, and a sleek web dashboard for mission control and AI research.
* **Tech Stack**: Python, FastAPI, React, Groq API, WebSocket.
